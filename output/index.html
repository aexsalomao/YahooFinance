Quotes.YahooFinance

Quotes.YahooFinance is a library created for accessing historical stock related data from the Yahoo! Finance database.

You can use Quotes.YahooFinance in dotnet interactive notebooks in Visual Studio Code or Jupyter, or in F# scripts (.fsx files), by referencing: #r "nuget: FSharp.Data" and #r "nuget: Quotes.YahooFinance" // Use the latest version

open System
open Quotes.YahooFinance

Downloading stock quotes

You can use YahooFinance.History to download the ticker history of a single stock or multiple stocks at once.

Parameters:

  1. symbols, required.

    A collection of ticker symbols.

  2. startDate, optional. Default is one year ago - DateTime.Today.AddYears(-1)

    The start date.

  3. endDate, optional. Default is today's date - DateTime.Today:

    The end date.

  4. interval, optional: Dafault is daily data - Interval.Daily

    Data granularity. To view the different types of intervals supported, refer to the Interval type documentation.

  5. displayLogs:

    Option to display logs.

Examples:

YahooFinance.History(symbols = "AAPL", 
                     startDate = DateTime.Today.AddMonths(-3), 
                     endDate = DateTime.Now, 
                     interval = Interval.Monthly,
                     displayLogs = true)
